# פרויקט מיני — מערכת קופה לקולנוע

תרגול משולב: **משתנים** (`var`/`let`/`const`), **תנאים** (`if/else`, `switch`, truthy/falsy) ו**לולאות** (`for`, `while`, `break`, `continue`)

---

## על הפרויקט

אתם בונים תוכנית קטנה שמדמה קופה של קולנוע. התוכנית:

1. שומרת רשימת לקוחות שרוצים לקנות כרטיס
2. בודקת לכל לקוח אם הוא יכול להיכנס לסרט (לפי גיל וסוג הסרט)
3. מחשבת לו מחיר כרטיס (עם הנחות)
4. מסכמת בסוף כמה כרטיסים נמכרו וכמה הכנסה התקבלה

---

## חלק 1 — משתנים (Variables)

צרו את המשתנים הבאים בתחילת הקובץ, עם הסוג הנכון (`const` או `let`):

```js
// מחיר בסיס לכרטיס
const BASE_PRICE = 45;

// גיל מינימלי לסרטי "12+" ו-"18+"
const AGE_LIMIT_12 = 12;
const AGE_LIMIT_18 = 18;

// משתנים שיצטברו / ישתנו במהלך התוכנית
let totalTicketsSold = 0;
let totalRevenue = 0;
```

**שאלה לחשיבה (תכתבו תשובה בקומנט מעל המשתנים):**
למה `BASE_PRICE` הוא `const` ואילו `totalRevenue` הוא `let`?

## חלק 2 — רשימת הלקוחות

הוסיפו מערך של אובייקטים שמייצגים לקוחות בתור לקופה:

```js
const customers = [
  { name: "דנה",  age: 17, movieRating: "12+", isStudent: true  },
  { name: "יוסי", age: 25, movieRating: "18+", isStudent: false },
  { name: "מיכל", age: 10, movieRating: "12+", isStudent: false },
  { name: "רון",  age: 16, movieRating: "18+", isStudent: true  },
  { name: "ליאת", age: 30, movieRating: "כל הגילאים", isStudent: false },
  { name: "אבי",  age: 8,  movieRating: "כל הגילאים", isStudent: false }
];
```

(אתם מוזמנים להוסיף עוד לקוחות בעצמכם בסוף)

---

## חלק 3 — לולאה שעוברת על כל הלקוחות

בנו לולאת `for` שעוברת על כל אובייקט במערך `customers`. עבור **כל** לקוח, התוכנית צריכה:

### א. לבדוק התאמת גיל (תנאים)


| תנאי                             | תוצאה                                                          |
| -------------------------------- | -------------------------------------------------------------- |
| הסרט `"18+"` והלקוח מתחת לגיל 18 | **לא נכנס** — להדפיס הודעת דחייה ולעבור ללקוח הבא (`continue`) |
| הסרט `"12+"` והלקוח מתחת לגיל 12 | **לא נכנס** — להדפיס הודעת דחייה ולעבור ללקוח הבא (`continue`) |
| כל מקרה אחר                      | **נכנס** — להמשיך לחישוב המחיר                                 |


> שימו לב — זה תרגיל מצוין לשימוש ב-`continue`: אם הלקוח נדחה, אין סיבה להמשיך לחשב לו מחיר.

### ב. לחשב מחיר לפי `switch` על `movieRating`

```js
let price = BASE_PRICE;

switch (customer.movieRating) {
  case "18+":
    price = BASE_PRICE + 10;
    break;
  case "12+":
    price = BASE_PRICE + 5;
    break;
  case "כל הגילאים":
    price = BASE_PRICE;
    break;
  default:
    price = BASE_PRICE;
}
```

### ג. להוסיף הנחות (תנאים + אופרטורים לוגיים)

- אם `isStudent === true` → הנחה של 15% מהמחיר
- אם הגיל **מעל** 65 → הנחה של 20% מהמחיר (גם אם הוא סטודנט — תבחרו רק את ההנחה הגבוהה יותר, לא לחבר את שתיהן)
- השתמשו ב-`ternary` כדי להחליט איזה הנחה לתת (אם בכלל)

### ד. לעדכן את הסכומים הכוללים

עבור כל לקוח שכן נכנס לסרט:

```js
totalTicketsSold++;
totalRevenue += price;
```

### ה. להדפיס שורת סיכום ללקוח

לדוגמה:

```
דנה (17) — כרטיס ל-12+ — מחיר: 50 ש"ח
מיכל (10) — לא עומדת בתנאי הגיל לסרט 12+
רון (16) — סטודנט — כרטיס ל-18+ — מחיר: 46.75 ש"ח (15% הנחה)
```

## חלק 4 — תנאי עצירה עם `break`

נניח שיש מגבלת מקומות באולם — **רק 4 כרטיסים** ניתן למכור בהקרנה הזו.

הוסיפו בתוך הלולאה בדיקה: אם `totalTicketsSold` הגיע ל-4, יש להדפיס `"האולם מלא! מפסיקים למכור כרטיסים."` ולעצור את הלולאה לחלוטין עם `break`.

> שימו לב להבדל בין המקום שבו תשתמשו ב-`continue` (חלק 3א) לבין המקום שבו תשתמשו ב-`break` (חלק 4) — זה בדיוק התרגול שרצינו.

---

## חלק 5 — לולאת `while` קטנה

לפני שמתחילים למכור כרטיסים, התוכנית צריכה "לפתוח את הקופה" — דמיינו שצריך לוודא שיש מספיק עודף בקופה.

```js
let cashInRegister = 0;
const MINIMUM_CASH = 200;

while (cashInRegister < MINIMUM_CASH) {
  cashInRegister += 50; // מוסיפים שקים של 50 ש"ח לקופה
  console.log("הוספנו 50 ש\"ח לקופה. סה\"כ בקופה: " + cashInRegister);
}

console.log("הקופה מוכנה לעבודה עם " + cashInRegister + " ש\"ח");
```

הריצו את הקוד הזה **לפני** הלולאה שעוברת על הלקוחות, והבינו לעצמכם — מתי הלולאה עוצרת ולמה.

---

## חלק 6 — סיכום בסוף התוכנית

לאחר שהלולאה על הלקוחות מסתיימת, הדפיסו סיכום:

```js
console.log("====================");
console.log("סה\"כ כרטיסים שנמכרו: " + totalTicketsSold);
console.log("סה\"כ הכנסה: " + totalRevenue + " ש\"ח");

const average = totalTicketsSold > 0 ? totalRevenue / totalTicketsSold : 0;
console.log("מחיר ממוצע לכרטיס: " + average.toFixed(2) + " ש\"ח");
```

שימו לב לשימוש ב-`ternary` כדי להימנע מחילוק באפס אם בטעות לא נמכר אף כרטיס.

## רשימת בדיקה לפני הגשה

ודאו שהתוכנית שלכם עומדת בכל הסעיפים:

- [ ] משתמשים נכון ב-`const` ו-`let` (אף שימוש ב-`var`)
- [ ] יש לפחות `if/else` אחד, `switch` אחד ו-`ternary` אחד
- [ ] נעשה שימוש נכון ב-`continue` (דילוג על לקוח) ו-`break` (עצירת מכירה)
- [ ] יש לולאת `for` אחת ולולאת `while` אחת
- [ ] כל ההשוואות נעשות עם `===` ולא `==`
- [ ] התוכנית רצה בלי שגיאות מתחילתה ועד סופה
